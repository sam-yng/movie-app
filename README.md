# Movie App Beta

---

## Setup
1. Clone the repo

`git clone [SSH or HTTPS]`

2. Install backend dependencies

`cd server`
`npm install`

3. Install frontend dependencies

`cd ../client`
`yarn install`

## Running the app
Backend:

`npm run dev`
Frontend:

`yarn dev`
