# Movie App Beta

---

## Setup
1. Clone the repo
```shell
git clone [SSH] or [HTTPS]
```

2. Install backend dependencies
```shell
cd server
```
```shell
npm install
```

3. Install frontend dependencies
```shell
cd ../client
```
```shell
yarn install
```

## Running the app
Backend:
```shell
npm run dev
```
Frontend:
```shell
yarn dev
```
